1. Data Cleansing Architecture

  • Outlier Rectification: Truncated age values containing negative constraints to logical >18 boundary maxing at 100.
  • Imputation Strategy: Adopted median imputation for skewed numerical properties and mode for topological categorical variables.
  • Target Alignment: Validated binary distribution ensuring unbiased 50/50 prior probabilities before cross-validation training.

3. Final Recommendations

The Random Forest model optimized via GridSearchCV (n_estimators=100, max_depth=8) produced the highest generalized AUC (0.8911) against the holdout set while mitigating overfitting observed in Gradient Boosting. The dominant signal vector lies in monetary transaction proxies (`total_spend_est`, `avg_order_value`), indicating predictive intent sits strongly in prior basket sizes. We recommend deploying the Random Forest pipeline downstream inside the CRM to allocate automated Platinum-marketing workflows to predictions scoring > 75% confidence limit.
